CAD · Systems Design · Aerospace
UAV System Design & Integration
Designed and integrated a fixed-wing UAV system — SolidWorks fuselage and wing modelling, aerodynamic trade-off analysis, and internal avionics wiring layout.

Technical Contributions

  • Designed system-level CAD models for a fixed-wing UAV fuselage and wings in SolidWorks, optimised for mission payload and range requirements
  • Executed design segmentation for additive manufacturing to reduce part count while maintaining structural integrity
  • Planned internal wiring routes for avionics and control surfaces, minimising electromagnetic interference
  • Performed conceptual aircraft sizing and aerodynamic trade-off analysis — wing loading, aspect ratio, lift-drag behaviour at low Reynolds numbers
  • Selected a high-lift airfoil by analysing trade-offs between aspect ratio and structural stiffness

Key Learning Outcomes

  • Translating mission requirements into geometric and structural constraints demands systematic trade-off analysis
  • Design for additive manufacturing requires planning assembly sequences prior to geometric modelling
  • Low-Reynolds-number aerodynamics deviates significantly from classical high-Re textbook assumptions

Python · Thermodynamics · Simulation
Supercharger Effects on IC Engine Performance
Developed Python models to simulate and visualise the thermodynamic impact of supercharging on IC engine performance — boost pressure analysis, volumetric efficiency, and performance curve generation.

Technical Contributions

  • Developed Python scripts to analyse and visualise supercharger impact on engine performance, creating performance curves for various boost pressures
  • Modelled thermodynamic parameters: air-fuel ratio, combustion efficiency, volumetric efficiency — achieving a simulated 134% volumetric efficiency
  • Analysed the influence of intake pressure and temperature rise on engine breathing characteristics
  • Generated boost pressure vs. power and efficiency vs. speed curves using Matplotlib and Pandas

Key Learning Outcomes

  • Supercharging fundamentally alters thermal behaviour and introduces new thermodynamic design constraints
  • Data visualisation is a primary engineering analysis tool, not a supplementary output
  • Implementing thermodynamic models in code requires rigorous understanding of governing equations
Python Matplotlib Pandas Thermodynamics IC Engine Theory

Python · Kinematics · Robotics
Kinematic System Design — Quadruped Robot
Designed and simulated the kinematic system of a quadruped robot in Python — forward and inverse kinematics, joint trajectory simulation, and gait pattern visualisation.

Technical Contributions

  • Derived and implemented forward and inverse kinematic models for a 4-legged robot leg system using NumPy
  • Simulated joint angle trajectories for multiple gait patterns
  • Visualised foot-path trajectories and joint workspace envelopes using Matplotlib

Key Learning Outcomes

  • Kinematic chain analysis requires careful frame assignment and Denavit-Hartenberg parameter definition
  • Gait design demands simultaneous consideration of mechanical limits and computational efficiency

Fluid Flow Analysis Over an Airfoil
Implemented a Python-based fluid flow simulation over an airfoil cross-section — pressure coefficient distribution, velocity contour mapping, and streamline visualisation.

Technical Contributions

  • Generated NACA airfoil geometry and applied panel method / potential flow analysis in Python
  • Computed and plotted pressure coefficient (Cp) distribution along the chord
  • Visualised streamlines and velocity contours around the airfoil profile using Matplotlib

Key Learning Outcomes

  • Numerical flow analysis bridges theoretical fluid mechanics and practical aerodynamic design
  • Small changes in angle of attack produce nonlinear, large-magnitude effects on pressure distribution
